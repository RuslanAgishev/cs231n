# cs231n
[CS231N](http://cs231n.stanford.edu/2019/syllabus.html) Stanford University programming assignments

## Installation
```bash
conda create -n cs231n python=3 anaconda
conda activate cs231n
```
### Assignment 1
```
cd assignment1
./install_packages.sh
cd cs231n/datasets
./get_datasets.sh
```
